Es un protocolo de seguridad estándar utilizado para establecer una conexión cifrada entre un cliente y un servidor en Internet. El SSL fue desarrollado por Netscape en la década de 1990 y se utiliza ampliamente para garantizar la seguridad de las comunicaciones en línea. El objetivo principal del SSL es proteger la confidencialidad y la integridad de los datos transmitidos a través de Internet. Utiliza técnicas de cifrado para encriptar los datos y prevenir que sean interceptados o alterados por terceros malintencionados. Cuando se establece una conexión SSL, el cliente y el servidor realizan un proceso de "apretón de manos" en el que acuerdan un conjunto de claves y algoritmos criptográficos para proteger la comunicación.

Burp suite es un escáner de vulnerabilidades web y una de las herramientas más conocidas y utilizadas para realizar auditorías de seguridad de aplicaciones web. Burp suite se sitúa en el medio, entre el navegador del usuario y la plataforma web o api móvil a auditar, registrando todo el tráfico, detectando vulnerabilidades de forma pasiva y permitiendo modificar las peticiones HTTP realizadas para detectar anomalías y problemas de seguridad. Desarrollada por la empresa europea Portswigger, burp suite es un referente por su tecnología de análisis de seguridad web, ya que complementa el análisis automático de vulnerabilidades con un potente framework con el que realizar pruebas manuales.

Un dominio, en el contexto de Internet, es una identificación única y legible por humanos que se utiliza para acceder a un sitio web específico. Consiste en dos partes principales: el nombre de dominio y la extensión de dominio. El nombre de dominio es la parte personalizada y distintiva del dominio que generalmente se elige para representar una entidad, como una empresa, organización o individuo. Por ejemplo, en el dominio "ejemplo.com", "ejemplo" es el nombre de dominio. La extensión de dominio, también conocida como TLD (Top-Level Domain), es la parte que sigue al nombre de dominio y generalmente indica la categoría o el propósito del sitio web. Algunas extensiones de dominio comunes son ".com" (comercial), ".org" (organización sin ánimo de lucro), ".net" (red), ".edu" (instituciones educativas) y ".gov" (gobierno). Sin embargo, existen muchas otras extensiones disponibles. Los dominios pueden registrarse a través de registradores de dominios, que son empresas autorizadas para administrar y asignar nombres de dominio.
